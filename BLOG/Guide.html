Being a profesional and respected programmer entails you follow the laid out protocols and steps in programming in other to prevent you fron becoming a quack programmer. The steps in programming include:

1. Program Specification

2. Program Design (UI/UX)

3. Program Code

4. Program Test

5. Program Documentation

6. Program Maintenance
